1. HTML – what to use it for

What is HTML?

HTML is the structure of a webpage. It stands for Hypertext Markup Language. This is a difficult word! What it means is that it’s just the way of preparing the website that any browser can read. For example, all the text on this website is programmed in HTML.

This picture might look a bit complicated, but this is the code of the page you’re currently looking at. Pretty cool, right?

2. HTML/ Tags & Attributes

As you can see in the photo, all lines begin and end with funny characters: <>. In between you can see p’s, h1’s, img’s. What does this mean?

These are called Tags. They tell the browser what type of text should be displayed. You can compare this to a book. A book contains chapters, titles, subtitles, paragraphs, images, etcetera.
